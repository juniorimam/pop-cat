## Run Locally

Clone the project

```bash
  git clone https://github.com/juniorimam/pop-cat
```

Go to the project directory

```bash
  cd pop-cat
```

Install dependencies

```bash
  npm install or npm i
```

Start the Pop Cat

```bash
  npm run dev
```
